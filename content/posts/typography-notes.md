見た目の統一よりも、読み心地の一貫性を優先する。字間、行間、段落間のリズムが揃えば、強い個性がなくても美しい。

```ts
const lineLength = 72;
const leading = 1.75;
const letterSpacing = "0.01em";
```

英数字は細め、和文はやや太め。太さの差で視線が引っかからないように調整する。
